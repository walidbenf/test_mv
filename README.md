# Game Search Platform

A video game search platform built with Next.js, TypeScript, and PostgreSQL, using the IGDB API.

## Technologies

- Next.js 14
- TypeScript
- PostgreSQL
- Prisma ORM
- IGDB API
- Tailwind CSS

## Prerequisites

- Node.js 18+
- PostgreSQL
- IGDB API credentials
